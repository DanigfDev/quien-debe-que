🔑 API Key de Anthropic

Necesaria para leer tickets con IA. Consíguela gratis en console.anthropic.com → API Keys.

Tu clave se guarda solo en tu navegador (localStorage). Esta app no tiene servidor propio — las llamadas van directamente desde tu navegador a Anthropic.

ℹ️ Cómo funciona el escaneo
Al subir una foto, la imagen se envía directamente desde tu navegador a la API de Anthropic usando tu clave. La app no tiene backend — nadie más ve tu clave ni tus fotos.

El modelo que se usa es claude-opus-4-5 (con visión). Cada escaneo consume una pequeña cantidad de crédito de tu cuenta Anthropic.
